# Tự kiểm tiếp cận AA

- [x] Cỡ chữ gốc 18px, hành động tối thiểu 18px.
- [x] Điều khiển có focus ring rõ và vùng bấm lớn.
- [x] Không chỉ dùng màu: thẻ luôn có chữ mức rủi ro.
- [x] Nhãn textarea, landmark, skip link, aria-live cho kết quả/lỗi.
- [x] Tương phản văn bản/nền theo bảng màu AA; có chế độ tương phản cao.
- [x] Bố cục một cột dưới 720px, không tràn ngang iPhone.
- [x] Phóng chữ và lưu lựa chọn.
- [ ] Kiểm tra VoiceOver và Safari trên iPhone thật trước khi nộp.
