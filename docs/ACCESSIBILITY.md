# Tự kiểm tiếp cận AA

- [x] Nội dung và điều khiển tối thiểu 18px; A+ tăng cỡ chữ gốc lên 21px.
- [x] Điều khiển có focus ring rõ và vùng bấm lớn.
- [x] Không chỉ dùng màu: thẻ luôn có chữ mức rủi ro.
- [x] Nhãn textarea, landmark, skip link, aria-live cho kết quả/lỗi.
- [x] Tương phản văn bản/nền theo bảng màu AA; có chế độ tương phản cao nền đen/chữ trắng riêng với giao diện sáng–tối.
- [x] Bố cục một cột dưới 720px; đã kiểm tra 360px và 360px + A+ không tràn ngang.
- [x] Phóng chữ và lưu lựa chọn.
- [ ] Kiểm tra VoiceOver và Safari trên iPhone thật trước khi nộp.
